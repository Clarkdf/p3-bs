This website works is as follows:

  1. User enters website
  2. User searches for a video game (or game franchise) in the search bar
  3. Website connects to RAWG video game database and searches for games (picks at most 25)
  4. The search results are displayed as buttons
  5. User presses button, site reconnects with the rawG database, pulls, and displays information about the game
  6. Website saves the game's title
  7. Website connects to the YouTube search API and uses the title to search for the game's soundtrack
  8. Website pulls the first result from the search API and embeds it on the bottom of the page
